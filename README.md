# FullStack CRUD Frontend

A simple CRUD (Create, Read, Update, Delete) frontend application built using React.

## Features

- Add new users/data
- Update existing data
- Delete data
- View all records
- Responsive UI

## Tech Stack

- React
- JavaScript
- CSS
- Axios

## Backend

This frontend is connected to a Spring Boot backend API.

## Installation

Clone the repository:

```bash
git clone https://github.com/ASHVIN-NEGI-07/FullStack-CRUD-Frontend.git
```

Go to the project folder:

```bash
cd FullStack-CRUD-Frontend
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
